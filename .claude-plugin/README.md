# 朝会・夕会アシスタント (Standup Assistant)

GitHub の作業状況を自動収集し、スクラムマスターとして朝会・夕会を進行する Claude Code スキルです。

## 機能

- 📝 **自動情報収集**: Git のコミット履歴、差分、PR情報を自動収集
- 🗣️ **対話的な進行**: スクラムマスターとして朝会・夕会を自然な会話で進行
- ⏰ **時間指定**: 過去 N 時間の情報を柔軟に指定可能
- 🔍 **詳細な分析**: コードレビューやブロッカーの発見をサポート

## 使い方

### 朝会（Morning Standup）

```bash
/standup morning
```

または自然言語で：
```
朝会を始めましょう
```

過去24時間の作業状況を確認し、今日の予定を立てます。

### 夕会（Evening Standup）

```bash
/standup evening
```

または：
```
今日の振り返りをしたい
```

過去10時間の作業を振り返り、日報を作成します。

### 時間をカスタマイズ

```bash
/standup morning 48    # 過去48時間
/standup evening 8     # 過去8時間
```

## 収集される情報

- ✅ コミット履歴（過去N時間）
- ✅ 変更の詳細（stat、差分）
- ✅ 未コミットの変更
- ✅ ステージ済みの変更
- ✅ オープン中の PR（自分が作成）
- ✅ 最近マージされた PR
- ✅ レビュー依頼されている PR

## 前提条件

### 必須

- **Git**: コミット履歴と差分の取得に使用
  - 通常、すでにインストール済みです

### オプション

- **GitHub CLI (`gh`)**: PR・Issue 情報の取得に使用
  - インストール: `brew install gh && gh auth login`
  - なくても基本機能（コミット履歴、差分）は動作します
  - PR情報のみ表示されません

**GitHub CLI のインストール**

```bash
# macOS
brew install gh
gh auth login

# Linux
# https://github.com/cli/cli/blob/trunk/docs/install_linux.md

# Windows
# https://github.com/cli/cli#installation
```

## 朝会・夕会の流れ

### 朝会（5〜10分）

1. 「おはようございます！」と挨拶
2. 昨日の作業を簡潔にまとめ
3. 今日の予定や目標を確認
4. ブロッカーや相談事を確認
5. アクションアイテムを整理

### 夕会（5分）

1. 「おつかれさまです！」と挨拶
2. 今日の作業を要約
3. 手応えや学びを確認
4. 明日への持ち越しを確認
5. 簡単な日報サマリーを作成

## Tips

- **音声モードと併用**: Claude の音声モードで、より自然な会話形式の朝会・夕会が可能
- **定期実行**: cron や Automator で定期実行すると便利
- **複数リポジトリ**: 複数のリポジトリで順番に実行可能
