# マーケットプレイス掲載ガイド

このスキルを各種マーケットプレイスに掲載する方法をまとめました。

## 🌟 SkillsMP (https://skillsmp.com)

### 自動掲載の仕組み

SkillsMP は GitHub リポジトリを自動的にスキャンして掲載します。

**必要な条件:**
- ✅ GitHub 公開リポジトリ
- ✅ `SKILL.md` ファイル（Agent Skills 標準形式）
- ✅ GitHub Topics に `claude-code-skill` を含む
- ⚠️ 最低 2 スター（品質フィルター）

**掲載までの流れ:**
1. GitHub にスキルを公開（済み）
2. Topics を設定（済み）
3. 2スター以上を獲得
4. SkillsMP のクローラーが自動検出（通常24〜48時間以内）
5. マーケットプレイスに掲載

### スターを増やす方法

1. **SNS で共有**
   ```
   🎉 Claude Code 向けの朝会・夕会スキルを公開しました！

   GitHub の作業状況を自動収集して、スクラムマスターとして
   朝会・夕会を進行してくれます。

   ⭐️ スターお願いします！
   https://github.com/t1k2a/claude-hurikaeri

   #ClaudeCode #AgileTools #Scrum
   ```

2. **Claude Code コミュニティで共有**
   - Discord サーバー
   - Reddit (r/ClaudeCode)
   - Twitter/X

3. **dev.to や Qiita で記事を書く**
   - 使い方の解説
   - 開発の背景
   - 実際の使用例

4. **GitHub の awesome リスト に追加**
   - [awesome-claude-skills](https://github.com/travisvn/awesome-claude-skills)
   - PR を送る

### 確認方法

掲載されたか確認:
```
https://skillsmp.com/search?q=standup
https://skillsmp.com/search?q=claude-hurikaeri
```

## 🏪 その他のマーケットプレイス

### SkillHub (https://www.skillhub.club/)

**特徴:**
- 7,000+ AI評価済みスキル
- Claude, Codex, Gemini, OpenCode 対応

**掲載方法:**
1. GitHub に公開（済み）
2. サイトで検索して確認
3. 必要に応じて手動申請

**申請先:** サイトの「Submit Skill」ボタンから（存在する場合）

---

### LobeHub Skills (https://lobehub.com/skills)

**特徴:**
- Agent Skills Marketplace
- マルチプラットフォーム対応

**掲載方法:**
1. GitHub リポジトリを公開（済み）
2. LobeHub で検索
3. 自動インデックスまたは手動申請

---

### MCP Market (https://mcpmarket.com)

**特徴:**
- MCP サーバー＆Agent Skills ディレクトリ
- Claude.ai, Claude Code, Codex, ChatGPT 対応

**掲載方法:**
1. GitHub リポジトリを公開（済み）
2. サイトで「Submit」オプションを探す
3. フォームから申請

## 🎯 優先アクション

**今すぐできること:**

1. **スターをお願いする**
   - 友人・同僚に共有
   - SNS で告知
   - コミュニティで紹介

2. **awesome-claude-skills に PR**
   ```bash
   # リポジトリをクローン
   git clone https://github.com/travisvn/awesome-claude-skills.git

   # READMEを編集してあなたのスキルを追加
   # PRを作成
   ```

3. **記事を書く**
   - Qiita/Zenn: 日本語コミュニティ
   - dev.to: 英語圏
   - Medium: 幅広い読者層

4. **48時間後に確認**
   - SkillsMP で検索
   - 掲載されていればシェア
   - されていなければ手動申請を検討
